ЧТО ДЕЛАТЬ ПРЯМО СЕЙЧАС (2 минуты):

1. Откройте: https://supabase.com/dashboard
2. Выберите ваш проект
3. Settings (⚙️) → API
4. service_role → Reveal → Copy
5. Откройте файл .env в корне проекта
6. Замените "your_service_role_key_here" на ваш ключ
7. Ctrl+C (остановите сервер)
8. npm run dev (запустите заново)

───────────────────────────────────────────────────────────────

ПРОВЕРКА:

1. Откройте: http://localhost:3000/test-connection
   → Должны быть ВСЕ зеленые галочки ✅

2. Откройте: http://localhost:3000/register
   → Зарегистрируйтесь (компания, email, пароль)
   → Вас перебросит на /app

3. Готово! ✨
