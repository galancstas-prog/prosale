### 4. Как работает переключение языка

**Текущее состояние:**
- По умолчанию приложение на **английском языке**
- Кнопка переключения языка **EN/RU** доступна:
  - В правом верхнем углу всех страниц приложения
  - На страницах входа и регистрации
- При нажатии кнопки **интерфейс мгновенно переключается** на выбранный язык
- Выбор языка **сохраняется в localStorage** браузера

**Доступные языки:**
1. **EN (English)** - английский
2. **RU (Русский)** - русский

## Технические детали

**Система переводов:**
- Файл с переводами: `lib/i18n/translations.ts`
- Контекст языка: `lib/i18n/use-locale.tsx`
- Компонент переключения: `components/locale-switcher.tsx`

**Добавление новых переводов:**
1. Откройте `lib/i18n/translations.ts`
2. Добавьте новый ключ в секцию `en:`
3. Добавьте перевод в секцию `ru:`
4. Используйте в компоненте: `t('your.new.key')`

**Пример:**
```typescript
// В translations.ts
en: {
  'mypage.title': 'My Page',
}
ru: {
  'mypage.title': 'Моя страница',
}

// В компоненте
import { useLocale } from '@/lib/i18n/use-locale'

function MyComponent() {
  const { t } = useLocale()
  return <h1>{t('mypage.title')}</h1>
}
```
